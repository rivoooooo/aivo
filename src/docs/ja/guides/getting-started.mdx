# はじめに

AI-Eraへようこそ。これはインタラクティブなチャレンジを通じて сучасних веб-розробкиスキルをマスターするためのプラットフォームです。

このガイドでは、最初のチャレンジの基本について説明します。

## 前提条件

開始する前に、以下を確認してください：

- 最新のWebブラウザ（Chrome、Firefox、Safari、またはEdge）
- AI-Eraアカウント
- HTML、CSS、JavaScriptの基本的な理解

### ステップ1：アカウントを作成

ホームページにアクセスし，注册ボタンをクリックします。詳細を入力してアカウントを作成してください。

### ステップ2：スキルマップを探索

スキルマップに移動して、利用可能なすべてのチャレンジを確認します。各ノードはあなたが学習できるスキルを表しています。

## 次のステップ

入門ガイドを完了したら、以下のチャレンジを試してください：

- 最初のチャレンジを完了する
- 異なるチャレンジカテゴリを探索する
- ダッシュボードで進捗を確認する
