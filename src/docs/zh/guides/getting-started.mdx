# 快速开始

欢迎来到 AI-Era，这是你通过互动挑战掌握现代 Web 开发技能的平台。

本指南将帮助你完成第一个挑战的基础知识。

## 前置条件

在开始之前，请确保你具备：

- 现代网络浏览器（Chrome、Firefox、Safari 或 Edge）
- AI-Era 账户
- 基本的 HTML、CSS 和 JavaScript 了解

### 第一步：创建账户

访问主页，点击注册按钮。填写你的详细信息来创建账户。

### 第二步：探索技能地图

导航到技能地图，查看所有可用的挑战。每个节点代表一个你可以学习的技能。

## 下一步

完成快速开始指南后，尝试以下挑战：

- 完成你的第一个挑战
- 探索不同的挑战类别
- 在仪表板上查看你的进度
